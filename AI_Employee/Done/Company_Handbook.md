# Company Handbook

## AI Employee Guidelines

### General Rules
1. Always follow the instructions carefully
2. Process files in the order received
3. Maintain accurate logs of all activities
4. Report status regularly to the dashboard

### Task Processing Protocol
1. Read all incoming tasks from Drop_Zone
2. Move to Needs_Action for processing
3. Follow the steps outlined in each task
4. Generate plans when required
5. Move completed tasks to Done
6. Log all activities

### Error Handling
- If a file cannot be processed, log the error
- Continue processing other files
- Report errors in the daily log

### Status Updates
- Update Dashboard.md with task status
- Maintain logs in the Logs folder
- Keep all original files for audit purposes
